二进制的概念

二进制表示：57

111001
2 * (6 - 1) + 2 * (5 - 1) + 2 * (4 - 1) + 2 * (1 - 1) = 32 + 16 + 8 + 1 = 57

十进制换二进制：短除2

除法         商     余数
57/2        28      1
28/2        14      0
14/2        7       0
 7/2        3       1
 3/2        1       1
 1/2        0       1
 将余数从下往上数，答案：111001

适合计算机的进制

二进制
    计算机内部用二进制：运算简单，简化了计算机结构
其他进制
    八进制：适用于12位和36位计算机系统
        标志的开头用0（零）表示，用0~7的数字表示
    16进制：用0~9,A,B,C,D,E,F表示
        表达长度短，变得更常用。标志的开头用0x表示

二进制位运算

运算符 名字 示例
& 与运算 6 & 3 = 2
| 或运算 6 | 3 = 7
^ 异或运算 6 ^ 3 = 5
~ 反码 ~6 = -7
<< 左移 3 << 2 = 12 (3 2 2 = 12)
>> 右移 3 >> 1 = 1 (3 / 2 = 1)
>>> 无符号右移 3 >>> 1 = 1 (3 / 2 = 1)

两变量交换值

1.借助第三个变量：C=A;A=B;B=C;
2.利用加减法：A=A+B;B=A-B;A=A-B;
3.异或运算实现：A=A^B;B=A^B;A=A^B;

数据类型 转化 字节

例：8143（00000000 00000000 00011111 11001111）
=> byte[] b = {-49,31,0,0};

第一个（低端）字节：8143 >> 0 * 8 & 0xff = (11001111) = 207(或有符号-49)
第二个（低端）字节：8143 >> 1 * 8 & 0xff = (00011111) = 31
第三个（低端）字节：8143 >> 2 * 8 & 0xff = (00000000) = 0
第四个（低端）字节：8143 >> 3 * 8 & 0xff = (00000000) = 0
  • 小端法（低端）：从低位开始存放入字节数组
  • 大端法（高端）：从高位开始存放入字节数组

字符串 -> 字节数组

String s = "abcde"; byte[] bs = s.getBytes();

字节数组 -> 字符串

byte[] bs = new byte[int];
String s = new String(bs);
或
String s  = new String(bs, encode);//encode指编码方式如：gb2312、utf-8
